## ADDED Requirements

### Requirement: Searchable Doubts List
The Doubts List SHALL include a top-level search bar.
- **Search Scope**: Title, Subject, and Topic.
- **Interaction**: The list SHALL update reactively as the user types.

### Requirement: Status Indication
The system SHALL display a high-visibility "Unanswered" badge for doubts where the last activity was a student post or where no mentor has responded.
