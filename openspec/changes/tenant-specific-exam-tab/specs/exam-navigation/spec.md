# Capability: Exam Navigation

## Purpose
Provides a dedicated entry point for exams, allowing users to view and manage their assessments from the primary navigation shell.

## ADDED Requirements

### Requirement: Exam Tab Entry Point
The system SHALL provide a dedicated "Exam" tab in the primary navigation shell.

#### Scenario: Navigating to Exams
- **WHEN** the user taps the "Exam" tab
- **THEN** the system SHALL display the Exams landing page showing available tests and assessments.
